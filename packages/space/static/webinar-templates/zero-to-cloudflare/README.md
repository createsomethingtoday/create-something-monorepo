# Automation Engineer · Step 1 Capstone

This is your first capstone on the path to becoming an Automation Engineer. Built at the CREATE SOMETHING workshop "Zero to Cloudflare in 30 Minutes."

## Quick Deploy

```bash
# Navigate to your projects folder
cd ~/projects

# Create and enter project folder
mkdir my-capstone && cd my-capstone

# Copy these files into the folder, then:

# Install Wrangler (Cloudflare's CLI)
npm install -g wrangler

# Login to Cloudflare
wrangler login

# Deploy to the edge
wrangler pages deploy .
```

Your capstone is now live globally.

## Terminal Commands Learned

| Command | What It Does |
|---------|--------------|
| `cd ~/projects` | Navigate to projects folder |
| `mkdir my-capstone` | Create a new folder |
| `cd my-capstone` | Enter the folder |
| `npm install -g wrangler` | Install Cloudflare CLI globally |
| `wrangler login` | Authenticate with Cloudflare |
| `cursor .` | Open Cursor in current folder |
| `wrangler pages deploy .` | Deploy current folder to edge |
