# Modern Dental Practice Template

A modern, clean dental practice website template with teal accent colors, patient testimonials, animated service cards, and appointment booking.

## Design

- **Primary Color**: Teal (#007a8a)
- **Typography**: Manrope (Google Fonts)
- **Icons**: Material Symbols Outlined
- **Dark Mode**: Full dark mode support

## Features

- Sticky navigation with backdrop blur
- Large hero section with patient testimonials overlay
- Service cards with hover animations
- Patient testimonials grid
- Full-width CTA section
- Newsletter signup in footer

## Quick Start

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build

# Deploy to Cloudflare Pages
pnpm deploy
```

## Configuration

Edit `src/lib/config/site.ts` to customize:

- Practice name and contact info
- Hero section content
- Services offered
- Patient testimonials
- CTA messaging

## Stack

- **Framework**: SvelteKit
- **Styling**: Tailwind CSS
- **Hosting**: Cloudflare Pages
- **Fonts**: Google Fonts (Manrope)
- **Icons**: Google Material Symbols
